# Exercícios

## Exercício 1 TMDB

No primeiro desafio, registrei-me no TMDB para obter uma chave de acesso API. Em seguida, utilizei essa chave para avançar na atividade proposta.

## Desafio parte 2

Na segunda etapa, dediquei tempo ao planejamento antes de começar a desenvolver os códigos. Após pesquisar sobre APIs e estudar como extrair informações de filmes usando o Lambda, iniciei o desenvolvimento no ambiente do VS Code. Foi necessário configurar camadas com bibliotecas específicas para garantir o funcionamento adequado do Lambda. Optei por coletar dados de filmes de drama e romance entre 2000 e 2024, relevantes para meu projeto final.

## Exercício 2 Massa de dados

1. Gerei uma lista de 250 inteiros aleatórios e os ordenei inversamente.
2. Compilei uma lista de 20 nomes de animais, ordenados alfabeticamente, e os armazenei em um arquivo CSV.
3. Utilizei a biblioteca "names" para gerar nomes de pessoas aleatórios, armazenando-os em um arquivo de texto.

## Exercício 3 Apache Spark

1. Preparei o ambiente e movi o arquivo "nomes_aleatorios.txt" para o diretório de desenvolvimento.
2. Importei as bibliotecas necessárias e iniciei a Spark Session.
3. Carreguei o arquivo CSV e exibi algumas linhas do DataFrame para visualização.
4. Renomeei colunas e imprimi o schema do DataFrame.
5. Adicionei aleatoriamente uma coluna de "Escolaridade" ao DataFrame.
6. Adicionei uma coluna "Pais" com nomes de países da América do Sul.
7. Incluí uma coluna "AnoNascimento" com valores aleatórios entre 1945 e 2010.
8. Seleccionei pessoas nascidas neste século utilizando métodos do DataFrame e Spark SQL.
9. Utilizei Spark SQL para contar pessoas por geração e país.

Este resumo aborda os principais passos e atividades realizadas durante o exercício TMDB, fornecendo uma visão geral do processo.
